# VC Glossary — 전체 용어집 (한국어/영어 대역)

크로스-스테이지로 쓰이는 VC 용어 전체를 카테고리별로 정리한다. 제품(UI 카피·spec·코드 식별자)에서 용어를 쓰거나 감수할 때 이 표를 기준으로 삼는다. **혼동하기 쉬운 용어 쌍과 실제 감수 절차는 `08-terminology-audit-checklist.md`** 참고.

## 1. 주체 (Personas & Parties)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 운용사 / 무한책임조합원 | GP (General Partner) | 펀드를 결성·운용하는 주체. 무한책임. **"운영사"(서비스·시설을 운영하는 회사)와 한 글자 차이 — 다른 말** |
| 출자자 / 유한책임조합원 | LP (Limited Partner) | 펀드에 출자하는 투자자(연기금·공제회·모태펀드·기업·개인). 유한책임 |
| 심사역 | Investment professional / Associate~Partner | 딜을 발굴·심사하고 투심위에 발제하는 VC 실무자 |
| 포트폴리오사 | Portfolio company | 펀드가 투자한 피투자기업 |
| 창업기획자 | Accelerator (AC) | 벤처투자법상 등록 액셀러레이터. 초기 투자+보육 |
| 자문사 | FA (Financial Advisor) | 딜을 중개·주선하는 자문사. 중개 딜(intermediated deal)의 출처 |
| 앵커 LP | Anchor LP | 결성 초기에 대규모 약정으로 펀드 신뢰도를 만들어주는 핵심 출자자 |

## 2. 펀드 구조·결성 (Fund Formation)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 벤처투자조합 | Venture investment fund | 벤처투자법상 결성하는 투자조합(펀드의 법적 그릇) |
| 결성총액 / 약정총액 | Fund size / Committed capital | LP+GP가 약정한 펀드 총 규모 = 총 출자좌수 × 1좌 금액 |
| 출자좌수 / 좌(구좌) | Units of contribution | 한국 조합형 펀드의 출자 기본 단위. 규약에 **"출자 1좌의 금액"** 을 정하고 LP/GP는 좌수로 약정. **의결권·배분은 좌수 비례**. 법령·규약 용어는 "좌"("구좌"는 구어). 미국식 펀드는 좌 없이 금액 약정 |
| GP 출자금 | GP commitment | GP가 자기 돈으로 넣는 출자(통상 1~5%) — skin in the game. 좌수로 표시 |
| 최소 출자금 | Minimum commitment | LP 1인당 최소 약정액(기관/개인 기준 상이) — 좌 단위 펀드에선 최소 인수 좌수 |
| 블라인드 펀드 | Blind pool fund | 투자 대상을 정하지 않고 전략만 정해 결성하는 펀드 |
| 프로젝트 펀드 | Project fund / Deal-by-deal SPV | 특정 딜을 정해두고 결성하는 펀드 |
| 규약 / 정관 | LPA (Limited Partnership Agreement) | 펀드의 헌법 — 전략·보수·권한·워터폴 규정 |
| 투자설명서 | PPM (Private Placement Memorandum) | LP 모집용 펀드 소개 문서 |
| 출자약정서 | Subscription agreement | LP의 출자 약정 계약 |
| 사이드레터 | Side letter | 특정 LP에게만 주는 개별 특약 |
| 1차 결성 / 최종 결성 | First closing / Final closing | 최소 결성액 도달 시 1차 클로징 후 추가 모집, 마감 시 파이널 클로징 |
| 출자지분율 | Percentage interest | 약정액(또는 좌수) ÷ 총액 — 캐피탈콜·배분·의결의 pro-rata 기준. 미국식 펀드의 몫 계산 축 |
| 이퀄라이제이션 | Equalization | 후속 클로징 참여 LP가 기존 LP의 기납입분+이자를 정산해 형평을 맞추는 절차 |
| 캐피탈 콜 / 수시납 | Capital call / Drawdown | 투자 집행 시점마다 약정 비율대로 납입 요청 (결성 시 전액 납입=일시납과 구분) |
| 존속기간 | Fund term | 통상 7~10년 = 투자기간 + 회수기간 |
| 투자기간 | Investment period | 신규 투자가 허용되는 기간(통상 3~4년) |
| 운용보수 | Management fee | 통상 연 2% 내외. 투자기간엔 약정총액, 이후 투자잔액 기준으로 체감 |
| 성과보수 | Carried interest (carry) | 초과이익의 GP 몫(통상 20%) |
| 목표수익률 | Target return (IRR) | GP가 PPM/IR에서 제시하는 성과 목표 — **구속력 없는** 계획 수치. hurdle(계약상 carry 문턱)과 다른 개념 |
| 기준수익률 / 우선수익률 | Hurdle rate (preferred return) | carry 발생의 최저 수익률 문턱(통상 연 7~8%, 국내 IRR 기준 다수) — **LPA/규약에 명시되는 계약 조항**으로 워터폴 계산에 직접 사용. Hard(초과분에만 carry)/Soft(catch-up으로 전체 이익에 carry) 구분 |
| 캐치업 | GP catch-up | hurdle 충족 후 GP가 carry 비율까지 따라잡는 배분 구간 |
| 클로백 | Clawback | 조기 배분된 carry가 최종 정산 시 초과로 판명되면 GP가 반환하는 조항 |
| 배분 순서 | Distribution waterfall | 원금 → hurdle → catch-up → carry 순의 회수금 배분 구조 (European=펀드 전체 기준 / American=딜별 기준) |
| 출자자 자문위원회 | LPAC | 주요 LP로 구성, 이해상충·규약 예외 승인 |
| 모태펀드 | Fund of funds | 펀드에 출자하는 펀드. 국내 대표: 한국벤처투자(KVIC) 운용 모태펀드 |
| 미소진 약정액 | Dry powder | 아직 집행하지 않은 투자 여력 |
| 빈티지 | Vintage year | 펀드 결성(첫 집행) 연도 — 성과 비교의 기준 축 |

## 3. 펀드 성과 지표 (Fund Performance)

| 용어 | 정의 |
|---|---|
| IRR (내부수익률) | 현금흐름 시점을 반영한 연환산 수익률. Gross(보수 차감 전)/Net(차감 후) 구분 필수 |
| MOIC / 투자배수 | 총가치 ÷ 투입원금. 시간 미반영 |
| DPI (실현배수) | 누적 분배액 ÷ 납입원금 — "실제로 돌려준 돈" |
| RVPI (미실현배수) | 잔여 NAV ÷ 납입원금 |
| TVPI (총가치배수) | DPI + RVPI |
| NAV (순자산가치) | 펀드/자산의 평가 기준 순자산 |
| J-커브 | 초기 보수·미실현 손실로 IRR이 음수였다가 회수기에 상승하는 곡선 |
| 밸류에이션 마크업/다운 | Mark-up / Mark-down — 후속 라운드 가격에 따른 보유지분 재평가 |

## 4. 딜 소싱·파이프라인 (Sourcing)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 딜 플로우 | Deal flow | 유입되는 투자 기회의 흐름/총량 |
| 딜 파이프라인 | Deal pipeline | 딜의 단계별 진행 현황 보드(소싱→심사→투심→계약→사후관리) |
| 직접 발굴 딜 | Proprietary deal | VC가 직접 소싱한 딜 — 경쟁 적고 조건 유리 |
| 중개 딜 | Intermediated deal | FA/IB 경유 딜 — 통상 경쟁 입찰 |
| 웜 인트로 | Warm intro | 신뢰 네트워크를 통한 소개 |
| 콜드 아웃리치 | Cold outreach | 사전 관계 없는 접촉(이메일 인바운드 포함) |
| IR / IR덱 | Investor Relations / Pitch deck | 스타트업의 투자유치 발표(자료) |
| 티저 / 원페이저 | Teaser / One-pager | 익명·요약형 딜 소개 자료 |
| 스크리닝 | Screening | 섹터·단계·체크사이즈 핏의 1차 필터 |
| 체크 사이즈 | Check size | VC의 1건당 투자 금액 범위 |
| 패스 | Pass | 투자하지 않기로 한 결정(거절) |

## 5. 펀딩 라운드 (Rounds)

| 용어 | 정의 |
|---|---|
| Pre-seed / Seed / Series A·B·C+ | 통상 12~18개월 간격의 라운드 명명. 트랙션/PMF 수준으로 구분 |
| 브릿지 라운드 | Bridge round — 본 라운드 사이를 잇는 소규모 조달(CB/SAFE가 흔함) |
| 업/플랫/다운 라운드 | Up/Flat/Down round — 직전 라운드 대비 밸류에이션 상승/동일/하락 |
| 익스텐션 라운드 | Extension — 같은 조건으로 기존 라운드를 증액 |
| 인사이드 라운드 | Inside round — 기존 주주만 참여하는 라운드 |
| 리드 투자자 | Lead investor — 조건 협상·실사를 주도하고 텀시트를 내는 투자자 |
| 팔로우 투자자 | Follower — 리드의 조건에 따라 참여 |
| 신디케이트 / 클럽딜 | Syndicate / Club deal — 복수 투자자의 공동 투자 구성 |
| 공동투자 | Co-investment — LP가 펀드 경유 외에 딜에 직접 참여 |

## 6. 투자 수단·증권 (Instruments)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 보통주 | Common stock | 기본 주식. 창업자·직원 보유분 |
| 전환우선주 | CPS (Convertible Preferred Stock) | 보통주 전환권이 붙은 우선주 |
| 상환전환우선주 | RCPS (Redeemable CPS) | 상환권+전환권이 붙은 우선주 — **한국 VC 투자의 사실상 표준 수단** |
| 전환사채 | CB (Convertible Bond/Note) | 주식 전환권이 붙은 사채 |
| 신주인수권부사채 | BW (Bond with Warrant) | 신주인수권(워런트)이 붙은 사채 |
| 조건부지분인수계약 | SAFE (Simple Agreement for Future Equity) | 밸류에이션을 후속 라운드로 유예하는 계약 — 벤처투자법으로 국내 법제화 |
| 밸류에이션 캡 | Valuation cap | SAFE/CB 전환 시 적용될 기업가치 상한 |
| 할인율 | Discount rate | SAFE/CB 전환 시 후속 라운드 가격 대비 할인 |
| 주식매수선택권 | Stock option | 임직원 스톡옵션의 국내 법정 용어 |
| 스톡옵션 풀 | Option pool (ESOP) | 임직원 배정용 예비 지분 — pre-money 포함 여부가 협상 포인트 |

## 7. 밸류에이션·지분 (Valuation & Cap Table)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 투자 전 기업가치 | Pre-money valuation | 투자금 유입 전 가치 |
| 투자 후 기업가치 | Post-money valuation | Pre-money + 신규 투자금. 지분율 = 투자금 ÷ post-money |
| 주주명부 / 지분표 | Cap table | 주주·증권별 지분 구조. **완전희석(fully diluted) 기준** 여부 명시 필수 |
| 완전희석 지분 | Fully diluted | 옵션·전환증권이 모두 전환됐다고 가정한 지분 |
| 희석 | Dilution | 신주 발행으로 기존 지분율이 낮아지는 것 |
| 현금흐름할인법 | DCF | 미래 FCF를 할인율(WACC)로 할인해 가치 산출 |
| 가중평균자본비용 | WACC | DCF의 할인율 |
| 잔존가치 | Terminal value (TV) | 예측기간 이후 가치의 현재가 환산분 |
| 상대가치평가 | Comparable company analysis (trading comps) | 동종업계 상장사 배수(PER·EV/EBITDA·EV/Revenue·PSR·PBR) 적용 |
| 유사거래비교 | Precedent transactions | 유사 M&A/투자 거래 배수 참조 |
| VC 방식 | VC method | 예상 Exit 가치를 목표수익배수로 역산 |
| 순자산가치법 | NAV approach | 자산-부채 기준 평가(자산형 딜·RWA에 적합) |

## 8. 실사·데이터룸 (Due Diligence)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 실사 | DD (Due diligence) | 투자 전 검증. 비즈니스/재무/법률 3단계 |
| 예비실사 / 본실사 | Preliminary / Confirmatory DD | 텀시트 전 개괄 검토 / 텀시트 후 확정 실사 |
| (가상) 데이터룸 | (Virtual) Data room, VDR | 실사 문서 공유 공간 |
| 단계적 공개 | Staged disclosure | 실사 진행도에 따라 공개 범위 확대 (관심→본실사→텀시트 후) |
| 실사 요청 목록 | DD request list | 투자자가 요구하는 문서 체크리스트 |
| Q&A | Q&A process | 데이터룸 내 질의응답 트랙 |
| 레드 플래그 | Red flag | 딜을 깨거나 조건을 바꿀 수 있는 발견 사항 |
| 이익의 질 검토 | QoE (Quality of Earnings) | 회계상 이익의 실질/지속성 검증 |
| 진술·보증 위반 | R&W breach | 계약상 진술보증과 실사 결과의 불일치 |

## 9. 트랙션 지표 (Startup Metrics — DD의 입력)

| 용어 | 정의 |
|---|---|
| MRR / ARR | 월간/연간 반복 매출 (구독형) |
| GMV / Take rate | 거래액 / 플랫폼 수수료율 (마켓플레이스형) |
| 번 레이트 / 런웨이 | Burn rate(월 순현금소진) / Runway(잔여 생존 개월수) |
| CAC / LTV | 고객획득비용 / 고객생애가치 — LTV/CAC 배수로 판단 |
| 리텐션 / churn | 유지율 / 이탈률. 코호트 기준(D30, M12 등) 명시 |
| NRR | 순매출유지율 — 기존 고객 매출의 확장-이탈 순효과 |
| PMF | Product-Market Fit |
| TAM / SAM / SOM | 전체·유효·수익화가능 시장 규모 |
| 유닛 이코노믹스 | Unit economics — 단위(고객/주문)당 손익 구조 |
| 인당 매출 | Revenue per headcount — 조직 효율 지표 |

## 10. 투심위·의사결정 (IC & Decision)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 투자심의위원회 (투심위) | IC (Investment Committee) | 투자 의사결정 기구(통상 임원급 3~5명). "투자심**사**위원회" 아님 |
| 투심 보고서 / IC 메모 | IC memo / Deal memo | 심사역의 발제 문서 — 논지·근거·리스크·추천 |
| 투자 논지 | Investment thesis | 이 딜이 성공하는 이유에 대한 반박 가능한 명제 |
| 조건부 승인 | Conditional approval | 특정 조건(추가 실사·조건 변경) 충족 전제의 가결 |
| 텀시트 | Term sheet | 주요 투자조건 요약 — **비구속(non-binding)**, 통상 배타조항·비밀유지만 구속 |
| 배타적 협상권 | Exclusivity / No-shop | 일정 기간 다른 투자자와 협상 금지 |
| 킬 조항 없음 유의 | — | 텀시트≠계약. 최종 구속력은 본계약(11절)에서 발생 |

## 11. 텀시트·투자계약 핵심 조항 (Key Terms)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 청산우선권 | Liquidation preference | 청산/매각 시 우선주가 먼저 회수할 권리. 배수(1x 등) + 참가적(participating)/비참가적 구분 |
| 희석방지조항 | Anti-dilution | 다운라운드 시 전환가격 조정. Full ratchet / 가중평균(broad·narrow) 방식 |
| 지분유지권 | Pro-rata right | 후속 라운드에서 기존 지분율만큼 참여할 권리 |
| 신주인수권(주주) | Preemptive right | 신주 발행 시 기존 주주의 우선 인수권 — BW의 워런트와 구분 |
| 우선매수권 | ROFR (Right of First Refusal) | 주주 지분 매각 시 우선 매수할 권리 (ROFO=우선협상권) |
| 동반매도참여권 | Tag-along (Co-sale) | 대주주 매각 시 같은 조건으로 함께 팔 권리 |
| 동반매도요구권 | Drag-along | 다수가 매각 시 소수 주주도 매각에 참여시키는 권리 |
| 사전동의권 / 보호조항 | Protective provisions (Consent rights) | 주요 경영사항에 대한 투자자 동의 요건 |
| 이사지명권 / 참관권 | Board seat / Observer right | 이사회 이사 지명 / 의결권 없는 참관 |
| 정보요청권 | Information rights | 재무·경영 정보의 정기 보고 요구권 |
| 상환권 | Redemption right | RCPS의 R — 일정 기간 후 원금(+이자) 상환 요구권 (배당가능이익 한도) |
| 위약벌 조항 주의 | Pay-to-play | 후속 라운드 불참 시 우선권 상실 |
| 창업자 지분 락업/베스팅 | Founder vesting / Lock-up | 지분의 시간 귀속(cliff=최소 근속 구간) / 매각 제한 |
| 주요 인력 조항 | Key man clause | 핵심 인력 이탈 시의 권리 발동 |
| 진술 및 보증 | Reps & Warranties | 회사·이해관계자의 사실 진술과 그 보증 |
| 손해배상 / 면책 | Indemnification | 진술보증 위반 시 배상 구조 |
| 선행조건 | CP (Conditions Precedent) | 납입(클로징) 전 충족돼야 하는 조건 |

## 12. 계약 문서 체계 (Definitive Agreements)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 신주인수계약 (투자계약) | SSA / Stock Purchase Agreement (신주) | **신주 발행·인수** 계약 — 신규 투자의 본계약 |
| 구주매매계약 | SPA (secondary) | **기존 주식** 양수도 계약 — 세컨더리 거래용. 신규 투자 문서를 "SPA"로 부르면 부정확 |
| 주주간계약 | SHA (Shareholders' Agreement) | 주주 간 권리(이사지명·동반매도·사전동의 등) 규율 |
| 투자자권리계약 | IRA (Investors' Rights Agreement) | (미국식) 정보권·등록권 등 — 국내에선 SHA에 통합되는 경우 多 |
| 정관 | Certificate of Incorporation / AoI | 우선주 종류·권리의 회사법적 근거 |
| 표준투자계약서 | Model documents | 미국 NVCA Model Legal Documents / 국내 KVCA·중기부 표준계약서 |
| 클로징 / 납입 | Closing | CP 충족 → 계약 체결·투자금 납입·주식 발행 완료 |
| 에스크로 | Escrow | 분쟁 대비 대금 일부 예치 |

## 13. 사후관리·Exit (Post-Investment)

| 한국어 | 영어 | 정의 |
|---|---|---|
| 사후관리 | Portfolio monitoring | 투자 후 KPI 수취·이사회 참여·지원 활동 |
| 보드덱 | Board deck | 이사회 보고 자료 |
| 후속투자 | Follow-on investment | 기존 포트폴리오사 추가 투자 |
| 리저브 | Reserve | 후속투자용으로 남겨두는 펀드 배분 |
| 감액 / 상각 | Write-down / Write-off | 보유지분 평가 하향 / 전액 손실 처리 |
| 회수 | Exit / Divestment | 투자금 회수 — M&A·IPO·세컨더리 |
| 기업공개 | IPO | 상장. 구주매출(기존 주주 매출) vs 신주모집 구분 |
| 보호예수 | Lock-up (IPO) | 상장 후 일정 기간 매각 금지 |
| 세컨더리 | Secondary sale | 지분을 다른 투자자에게 직접 매각 (세컨더리 펀드 포함) |
| 인수합병 | M&A / Trade sale | 전략적·재무적 인수자에 매각. Earn-out=성과 연동 후불 대금 |
| 청산 | Liquidation / Dissolution | 회사 정리 — 청산우선권 순서로 배분 |
